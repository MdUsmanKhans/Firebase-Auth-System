# Firebase Authentication

A simple Firebase Authentication project built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates how to implement **Email & Password Authentication** with Firebase, including user registration and login functionality.

## Features

* User Signup with Email & Password
* User Login with Email & Password
* Firebase Authentication Integration
* Responsive User Interface
* Clean and Beginner-Friendly Code

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Firebase Authentication

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/firebase-auth.git
```

### 2. Navigate to the Project Folder

```bash
cd firebase-auth
```

### 3. Create a Firebase Project

1. Open the Firebase Console.
2. Create a new project.
3. Enable **Email/Password Authentication**.
4. Register a Web App.
5. Copy your Firebase configuration.

### 4. Configure Firebase

Open **`js/firebaseConfig.js`** and replace the placeholder configuration with your own Firebase project configuration.

### 5. Run the Project

Open `signup.html` or `login.html` using Live Server or any local development server.
